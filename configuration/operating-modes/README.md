# Operating Modes

The operating mode of a link can be configured in the Edit window.

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

_Link Type_ options are

* [Display ](display/)- sending display data from one location to a remote display board from RemoteRedirect webservice to a remote display board
* [Pass Through](pass-through.md) - send data from one location to another location
* [Timing ](timing.md)- send timing data from one location to another location
